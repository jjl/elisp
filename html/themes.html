Auditory Icon Themes For Emacspeak

To install these themes, download and unpack the archive into directory emacspeak/sounds in your emacspeak instalation. You can select themes you have already installed by using Emacspeak command emacspeak-sounds-select-theme bound by default to control-e ).

Chimes
Theme made up mostly of different chimes and short notes from various instruments. The icons are high-quality 44K-mono; this is the theme I presently use.
Cartoons
This theme was contributed by Bryan Smart --and is made up of 22K-mono sounds.
